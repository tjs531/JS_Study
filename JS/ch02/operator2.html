연산자(operator)

typeof : 변수의 데이터타입을 확인하는 연산자
3.비교연산자 : >, >=, <, <=, ==(equal), !=(not equal), ===, !==
- ==, != : 값만 비교해서 true, false를 출력
- ===, !== : 값과 데이터타입을 모두 비교해서 true 또는 false를 출력
- 우선순위 : (>, >=, <, <=) 연산자가(==, !=)보다 우선임.
- 우선순위 : (==, !=) 연산자가(===, !==)보다 우선임.
- 우선순위 : (>, >=, <, <=) > (==, !=) > (===, !==)
4. 논리연산자 : &&(논리 AND, 논리곱), ||(논리OR, 논리합), !(논리NOT, 논리 부정)
-논리연산자의 결과도 논리값(true, flase 둘중의 하나);
-비교연산자가 논리연산자보다 우선임. 비교 > 논리
-논리AND가 논리OR보다 우선임.
-우선순위 : ! > && > ||
5.대입연산자 : =
복합대입연산자(할당연산자, assignment) : +=, -=, *=, /=, %=
